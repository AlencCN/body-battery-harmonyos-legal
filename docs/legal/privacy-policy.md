# 全面健康管理助手隐私政策

更新日期：2026-07-16  
生效日期：2026-07-16

“全面健康管理助手”（以下简称“本应用”）由【个人开发者姓名/主体名称】（以下简称“我们”）提供。本隐私政策说明我们如何收集、使用、存储、保护和处理你的个人信息，尤其是运动健康数据。

请在使用本应用前仔细阅读本隐私政策。若你不同意本隐私政策，请不要授权本应用读取你的健康数据。

## 一、我们的基本原则

1. 最小必要：仅在实现功能所必需的范围内读取数据。
2. 主动授权：仅在你明确授权后，通过 HarmonyOS Health Service Kit 读取相关健康数据。
3. 本地优先：当前版本以本地展示和本地分析为主，不主动向我们的服务器上传你的健康数据。
4. 可撤回：你可以随时在系统设置、华为运动健康或相关授权管理页面撤回授权。
5. 非医疗用途：本应用仅用于生活健康管理，不用于疾病诊断、治疗、处方或医疗决策。

## 二、我们处理的个人信息类型

### 1. 你主动授权读取的运动健康数据

在你授权后，本应用可能读取以下数据：

| 数据类型 | 读取内容示例 | 使用目的 |
| --- | --- | --- |
| 日常活动数据 | 步数、热量、距离、活动时长 | 展示每日活动量、活跃趋势和健康概览 |
| 锻炼记录概要 | 运动类型、开始/结束时间、总时长、消耗等 | 统计运动频率、训练时长和运动负荷 |
| 锻炼记录详情数据 | 单次运动过程数据、训练详情 | 展示运动详情、强度变化和训练完成情况 |
| 体脂数据 | 体重、BMI、体脂率等体成分字段 | 展示体成分趋势和健康管理看板 |
| 心率数据 | 全天心率、夜间心率、静息心率 | 展示心率趋势、恢复状态和运动强度参考 |
| 压力数据 | 压力分数、高压力时段 | 展示压力趋势和休息建议 |
| 睡眠数据 | 睡眠时长、深睡、REM、清醒次数等 | 展示睡眠恢复、睡眠结构和长期趋势 |
| 血压数据 | 收缩压、舒张压 | 展示个人血压记录和趋势 |
| 血氧数据 | 血氧饱和度 | 展示血氧趋势和恢复参考 |
| 情绪数据 | 情绪状态 | 展示情绪趋势及其与睡眠、压力、活动的关系 |
| 历史数据 | 一周、一月、一年的授权数据 | 生成长期趋势、个人基线和阶段性复盘 |

上述数据可能属于敏感个人信息。我们会在取得你的授权后处理，并仅用于本政策列明的目的。

### 2. 设备和运行信息

当前版本不主动采集设备标识、广告标识、精确定位、通讯录、短信、相册、麦克风、摄像头等信息。为保证本地功能运行，应用可能在设备本地读取必要的系统状态或运行状态，但不会主动上传到我们的服务器。

如果后续接入崩溃分析、云同步、账号系统、统计分析或其他网络服务，我们会在收集前更新本隐私政策，并在必要时重新征得你的同意。

### 3. 我们不处理的信息

当前版本不申请、不读取、不处理以下信息：

1. 锻炼记录位置详情数据。
2. 体温数据。
3. 生殖健康数据。
4. 通讯录、短信、通话记录、相册、摄像头、麦克风。
5. 广告追踪标识。
6. 健康数据写入权限。

## 三、我们如何使用个人信息

我们仅将你的授权数据用于以下目的：

1. 展示你的健康数据、运动记录和历史趋势。
2. 生成睡眠恢复、活动负荷、压力趋势、心率趋势等生活健康管理信息。
3. 在数据缺失或授权不足时进行功能降级展示。
4. 改善本地体验、排查本地功能异常。
5. 满足法律法规、平台审核或安全合规要求。

我们不会将你的健康数据用于广告画像、保险核保、信贷评估、就业评估、医疗诊断或向第三方出售。

## 五、数据存储与保留

1. 当前版本以本地处理为主，健康数据主要保存在你的设备本地或由系统/华为运动健康生态保存。
2. 本应用可能在设备本地缓存必要的展示数据、授权状态或页面状态，以便改善使用体验。
3. 你可以通过删除应用、清除应用数据或撤回系统授权来清除本应用本地数据或停止继续读取。
4. 若未来提供云同步或服务器功能，我们会说明服务器所在地、保存期限和删除方式，并在必要时重新征得你的同意。

## 六、数据共享、转让与公开披露

当前版本不会主动向第三方共享、转让或公开披露你的健康数据，以下情形除外：

1. 获得你的单独同意。
2. 法律法规、司法机关、行政机关依法要求。
3. 为保护你或他人的生命、财产等重大合法权益所必需。
4. 为实现你主动请求的功能，且你已明确知悉并同意相关第三方处理规则。

当前版本不接入广告 SDK、第三方统计 SDK 或第三方营销 SDK。

## 七、第三方服务说明

本应用接入 HarmonyOS Health Service Kit 以读取你授权范围内的运动健康数据。相关系统能力和数据来源可能由华为账号、设备系统、华为运动健康及 Health Service Kit 提供。你应同时阅读并遵守华为相关服务的协议和隐私政策。

如果后续新增第三方 SDK 或服务，我们会在本政策中列明名称、用途、处理的信息类型和退出方式。

## 八、你的权利

在法律法规允许范围内，你可以：

1. 查询、更正、复制你在本应用中展示或本地保存的个人信息。
2. 删除本应用本地保存的数据。
3. 撤回 Health Service Kit 相关授权。
4. 拒绝非必要数据处理。
5. 注销或停止使用本应用；当前版本不提供独立账号体系，因此删除应用即可停止使用。
6. 通过本政策提供的联系方式向我们提出个人信息相关请求、投诉或建议。

我们将在收到你的请求后，在法律法规要求的期限内处理。

## 九、未成年人保护

本应用不面向未成年人提供专门服务。若你是未成年人，应在监护人同意和指导下使用本应用。监护人如发现未成年人未经同意使用本应用或授权健康数据，可通过本政策联系方式联系我们处理。

## 十、信息安全

我们会采取合理的技术和管理措施保护你的个人信息，包括最小权限、授权校验、本地优先、避免不必要上传等措施。但请理解，任何系统都无法保证绝对安全。请妥善保管你的设备、系统账号和华为账号。

## 十一、跨境传输

当前版本不主动将你的个人信息传输至中国境外。如未来涉及跨境传输，我们会按照适用法律法规履行告知、同意和合规手续。

## 十二、政策更新

我们可能因法律法规、平台要求或产品功能变化更新本隐私政策。更新后，我们会在应用内、GitHub 页面或其他合理位置展示新版本。若更新涉及敏感个人信息处理目的、处理方式、共享对象等重大变化，我们会在必要时重新征得你的同意。
